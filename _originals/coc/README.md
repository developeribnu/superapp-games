# 🏰 CoC Knowledge Base

**Clash of Clans All in One** – Knowledge base personal untuk menyimpan semua pengetahuan tentang CoC.

## Fitur

- 📚 **Artikel terorganisir** berdasarkan kategori (Troops, Heroes, Spells, Buildings, Strategy, dll)
- 🔍 **Pencarian real-time** dengan highlight hasil pencarian dan autocomplete
- 🏷️ **Filter kategori** dengan chip yang mudah diklik
- ✍️ **Tambah & Edit artikel** dengan editor Markdown built-in
- 👁 **Preview Markdown** sebelum menyimpan
- ⭐ **Featured articles** untuk konten terpenting
- 📊 **View counter** untuk setiap artikel
- 🔃 **Sort** berdasarkan terbaru, terlama, A-Z, popularitas
- ⊞ **Grid & List view** toggle
- 💾 **Auto-save** ke localStorage (data tersimpan di browser)
- ⌨️ **Keyboard shortcuts**: `Ctrl+K` (search), `Ctrl+N` (artikel baru), `Esc` (tutup modal)
- 📱 **Responsive** – bekerja di mobile, tablet, dan desktop

## Cara Menggunakan

1. Buka `index.html` di browser
2. Klik **+ Tambah Artikel** untuk menambah pengetahuan baru
3. Tulis konten dalam format **Markdown**
4. Gunakan **Search** untuk mencari artikel
5. Filter berdasarkan **kategori** menggunakan chip

│   └── js/

## Tech Stack

- **Pure HTML/CSS/JavaScript** – tanpa framework, tanpa build process
- **LocalStorage** – untuk menyimpan artikel secara lokal di browser
- **Markdown** – format penulisan konten yang mudah
- **Google Fonts (Cinzel + Lato)** – untuk tampilan medieval CoC
